Estrutura básica do HTML

Você sabia que saber a estrutura básica do HTML é importante para o SEO e a boa experiência do usuário que acessar nosso site?

Oi pessoal, meu nome é Léo Vitor e nesse artigo vamos explorar para que serve cada linha e seus respectivos significados!

Se gostar desse artigo e quiser me dar uma ajudinha eu lhe recomendo ir a uma das minhas redes abaixo ;)

DOCTYPE o que é e para que serve?

A tag doctype como o próprio nome já diz é uma tag que define o tipo de conteúdo em que você está lidando.

Ou seja, quando definimos essa tag e usamos o parâmetro html estamos dizendo ao navegador que aquele tipo de documento é HTML para melhor interpretar quando for renderizado.

html o que é e para que serve?

A tag html é uma introdutoria do conteúdo do documento, ela é necessária para definir que o conteúdo dentro dela vai ter tags html.

Este atributo lang é usado para definir o tipo de tradução em que a página HTML vai ter, se não definir o navegador não saberá qual é a tradução do conteúdo e vai ter esta base no conteúdo existente na página.

Vale lembrar que está tag possui fechamento, diferente da tag principal da estrutura básica do HTML, essa tag se prolonga até o fim do documento.

head o que é e para que serve?

A tag head é como se fosse a cabeça do documento HTML, ela define a parte em como o navegador irá entender sua página.

É nela que ficará os metadados como o título da página, a descrição, a forma em que o navegador vai entender o conteúdo, quem fez o site, como funcionará o site em outros tamanhos, dentre outros, ela possuí fechamento.

meta o que é e para que serve?

As tags meta ajudam a "falar" ao navegador que for renderizar o documento HTML entender o que tem no conteúdo da página, isso é, também ajuda os mecanismos de busca entender o que o site se refere.

Na primeira linha, temos a definição do texto que vamos trabalhar que no caso é o UTF-8, ele é uma padronização para melhor o navegador entender cada caracter e converter para linguagem binária. Resumindo, ela ajuda para não ter bugs de acentuação, e fazer o navegador entender o texto que foi inserido no documento.

Na segunda linha, define que o documento HTML tenha compatibilidade com o Internet Explorer, antigo navegador muito utilizado, mas atualmente perdeu sua força, e a tag foi menos utilizada, tanto é que: Eu mesmo não utilizo rsrs.

Na terceira linha, define que o site irá se adaptar conforme a largura da tela do dispositivo, como sua escala inicial. Sem ela, o site poderá ficar desproporcional em dispositivos menores, ou muito maiores. Eu recomendo que você utilize sempre!.

title o que é e para que serve?

A tag title é basicamente o título do site, é ela que ficará vísivel na aba da página.

body o que é e para que serve?

A tag body é onde define o corpo da página HTML, é nela onde você colocará outras tags como h1, h2.. p, a, img, audio, video, dentre outras.

Simples de entender né? É na tag body que você colocará todo o conteúdo da sua página.
